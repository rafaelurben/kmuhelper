KMUHelper

Eine Buchhaltungssoftware für alle, welche sich keine professionelle Buchhaltungssoftware leisten können oder von überall auf ihr Geschäft zugreifen wollen. Einfach und simpel.

django-kmuhelper

Features

Der KMUHelper ist mit folgenden Features und noch vielem mehr ausgestattet.

Bestellungen

Erstelle Bestellungen, importiere sie von anderen Quellen und drucke sie aus.

QR-Rechnungen

Mit dem KMUHelper können QR-Rechnungen als PDF-Datei via Knopfdruck erstellt werden.

Online

Dadurch, dass der KMUHelper auf einem Server gehostet wird, ist er von überall erreichbar.

Kundenverwaltung

Erfasse deine Kunden und sende ihnen E-Mails mit ihrer personalisierten Rechnung.

Lagerverwaltung

Verwalte dein Lager und sehe sofort, wenn der Lagerbestand eines Produktes knapp wird.

Statistiken (Beta)

Sieh dir an, was deine Kunden in einem Zeitraum am meisten bestellt haben.
