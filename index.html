KMUHelper

Eine Bestellungsverwaltungssoftware mit Schweizer QR-Rechnungen für alle, welche sich keine professionelle Software leisten können oder von überall auf ihr Geschäft zugreifen wollen. Einfach und simpel.

django-kmuhelper - Handbuch

Features

Der KMUHelper ist mit folgenden Features und noch vielem mehr ausgestattet.

Bestellungen

Erstellen Sie Bestellungen, importieren Sie sie von anderen Quellen und drucken Sie sie aus.

QR-Rechnungen

Mit dem KMUHelper können QR-Rechnungen als PDF-Datei auf Knopfdruck erstellt werden.

Online

Dadurch, dass der KMUHelper auf einem Server gehostet wird, ist er von überall erreichbar.

E-Mails

Senden Sie E-Mails mit Anhängen und erstellen Sie Vorlagen.

XML Import

Importieren Sie camt-Dateien für einfache Zahlungsverarbeitung.

Kundenverwaltung

Erfassen Sie Ihre Kunden und senden Sie ihnen E-Mails mit ihrer personalisierten Rechnung.

Lagerverwaltung

Verwalten Sie Ihren Lagerbestand und sehen Sie sofort, wenn der Lagerbestand eines Produktes knapp wird.

Statistiken (Beta)

Sehen Sie sich an, was Ihre Kunden in einem Zeitraum am meisten bestellt haben.
